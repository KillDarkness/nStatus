nStatus API

Documentação completa da API para monitoramento de servidores Minecraft. Consulte status, jogadores, whitelist, banimentos e muito mais.

Introdução

A API nStatus permite que você monitore e gerencie seu servidor Minecraft remotamente através de requisições HTTP simples. Você pode verificar status do servidor, gerenciar jogadores, whitelist, banimentos, TPS e muito mais.

Esta documentação fornece todas as informações necessárias para começar a usar a API, incluindo endpoints disponíveis, parâmetros, exemplos de requisições e respostas.

Autenticação

A API nStatus pode exigir autenticação por token para proteção contra acessos não autorizados. A configuração de autenticação é definida no arquivo config.yml do plugin.

Como autenticar

Para autenticar, adicione o parâmetro token à sua requisição:

http://seu-ip:porta/status?token=seu-token

Status do Servidor

Consulte informações gerais sobre o status do seu servidor Minecraft.

Status Geral do Servidor

GET /status

Descrição

Mostra a quantidade de jogadores online e o máximo permitido no servidor.

Parâmetros

  • token (opcional): Token de autenticação

Exemplo de Requisição

GET http://seu-ip:porta/status?token=seu-token

Resposta

{
  "online": 5,
  "max": 20,
  "status": 200
}

Códigos de Status

  • 200: Requisição bem-sucedida
  • 401: Token de autenticação inválido ou ausente
  • 500: Erro interno do servidor

Status de um Jogador

GET /status/:player

Descrição

Mostra informações sobre um jogador específico, incluindo seu status, saúde e mundo atual.

Parâmetros

  • :player: Nome do jogador
  • token (opcional): Token de autenticação

Exemplo de Requisição

GET http://seu-ip:porta/status/Notch?token=seu-token

Resposta

{
  "player": "Notch",
  "online": true,
  "health": 20.0,
  "world": "world",
  "status": 200
}

Parâmetros:

  • :player - Nome do jogador

Respostas possíveis:

  • 200 - Jogador encontrado
  • 400 - Nome do jogador não especificado
  • 404 - Jogador não encontrado ou offline

Exemplos de Código

Confira exemplos de como usar a API nStatus em diferentes linguagens de programação.

// Exemplo de como consultar o status do servidor usando JavaScript
fetch('http://seu-ip:porta/status?token=seu-token')
  .then(response => response.json())
  .then(data => {
    console.log(`Jogadores online: ${data.online}/${data.max}`);
  })
  .catch(error => console.error('Erro:', error));
# Exemplo de como consultar o status do servidor usando Python
import requests

url = 'http://seu-ip:porta/status'
params = {'token': 'seu-token'}

response = requests.get(url, params=params)
data = response.json()

print(f"Jogadores online: {data['online']}/{data['max']}")
